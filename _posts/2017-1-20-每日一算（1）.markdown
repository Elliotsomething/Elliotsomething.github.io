### 牛顿迭代算法

**题：不用库函数，编写一个函数，求整数N的开方**

```objective_c
#define  ABS(VAL)   ((VAL>0)?(VAL):(-VAL))
double sqrt(float x)
{
    double g0,g1;
    g0=x/2;
    g1=(g0+x/g0)/2;
    while(ABS(g0-g1)>0.001)
    {
        go=g1;
        g1=(g0+x/g0)/2;
    }
    return g1;
}
```

这题算是网上的标准解法了，拿来练下手；
